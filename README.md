
# 🌍 Oman Vision 2040 Smart Agriculture AI

## Description
This project uses Machine Learning to recommend the best crops for different regions in Oman based on climate and soil conditions.

## Features
- Crop recommendation system
- Oman climate integration
- Top 5 crop ranking
- Suitability scoring

## Tech Stack
- Python
- Scikit-learn
- Streamlit
- Pandas

## Goal
Support Oman Vision 2040 for food security and smart agriculture.
